Frosted Glass Effect Demo

The frosted glass effect that you see to the right is created using a combination of CSS properties, the most prominent being backdrop-filter: blur(10px); or -webkit-backdrop-filter: blur(10px); which applies a blur to the background of the element.

The background of the element consists of a pair of fills assigned to its background-image stylesheet property. The first is a circular radial-gradient which is largely transparent except for a patch of 'bloom' near the top. The second uses a linear-gradient with near the extremes of the top and bottom of the container. This second background is allowed to show by using box-sizing: border-box to clip the container's main content to the inner edge of the border.

Finally, fairly hefty values are applied to the style's box-shadow property to give a very diffuse shadow around the element.

The header is positioned absolutely, 20px from the bottom-right of the container and uses a font called "Parisienne" linked to Google's font repository found here, whilst the body text uses Veranda or the closest available sans-serif font.

The <code> elements inside the element use a very similar set of styling to the main container, although the effect values are greatly reduced to suit the nature of the content.

A demonstration of frosted glass effect using Cascading Style Sheets.

The main <div> element used to create the box makes use of two background layers - a radial gradient to add sheen to the body, a linear gradient for the border and a blur backdrop filter. The heading is positioned absolutely at the bottom right of the box and uses the CSS property text-shadow to give a slightly diffuse effect to the font.

The body uses a sans-serif font (Verdana, Geneva, Tahoma, sans-serif) and the header uses "Parisienne", cursive from Google Fonts.

<code> elements inside the box also have a slight cumulative blur effect and linear gradient border and shadow to match the overall frosted glass appearance, although the overall effect of these has been lessened to avoid making them contrast too heavily against the rest of the content, whilst still allowing the background to show through.
